# Elysium

Provide a base for creating new React apps.

## Getting started

### Prerequisites

- Node [https://nodejs.org/en/](https://nodejs.org/en/)
- Yarn [https://yarnpkg.com/en/](https://yarnpkg.com/en/)

### Setup

Install all the dependencies

```sh
make setup
```

Run the development server

```sh
make run
```
